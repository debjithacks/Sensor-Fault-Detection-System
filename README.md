# Sensor Fault Detection System

A machine learning-powered web application for detecting faults in various sensor types including Wafer, Gas, Temperature, Soil Moisture, and Light sensors.

## Features

- **Multi-Sensor Support**: Detect faults across 5 different sensor types
- **Auto-Detection**: Automatically identify sensor type from uploaded data
- **User Authentication**: Secure login and signup system
- **Admin Dashboard**: Monitor user activities, download datasets, and export reports
- **Real-time Predictions**: Instant fault detection results
- **Data Export**: Download prediction results and activity reports

## Deployment
